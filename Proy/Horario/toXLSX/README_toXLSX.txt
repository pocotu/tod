# Script de Conversión de Archivos .xls a .xlsx

Este script de Visual Basic for Applications (VBA) se utiliza para convertir archivos .xls a .xlsx en Microsoft Excel.

## Instrucciones de Uso

Sigue estos pasos para ejecutar el script de conversión:

1. Abre el archivo.xls en Microsoft Excel.
2. Presiona ALT + F11 para abrir el Editor de Visual Basic.
3. En el apartado "VBAProject(FUNCRES.XLAM)", anticlic, baja hasta la opción (Insertar/Módulo).
4. Pega el código del archivo toXLSX.txt en ese espacio en blanco.
5. Presiona F5 y se te pedirá la ruta de la carpeta del archivo.xls; selecciónala.
6. Luego, selecciona la ruta de la carpeta donde se colocarán los nuevos archivos .xlsx.
